# TexPick

> 필요한 글만 고르고, 군더더기는 남기지 마세요.

복사한 채팅에서 닉네임, 시간, 반응 메뉴 같은 부가 정보를 제거하고 메시지 본문만 추출하는 개인정보 보호형 웹 도구입니다.

[실행 중인 서비스](https://chatextractor.chattools.workers.dev) · [영문 README](../README.md)

![TexPick 화면](screenshot.png)

## 왜 만들었나요?

채팅 앱에서 대화를 복사하면 본문뿐 아니라 닉네임, 날짜, 시간, 이모티콘 반응, 답장 버튼, 채널 안내 문구까지 함께 따라옵니다. 다른 문서나 메모에 정리하려면 매번 직접 지워야 합니다.

TexPick은 이 과정을 자동화합니다. 붙여 넣은 내용은 서버로 전송하지 않고 사용자의 브라우저 안에서만 처리합니다.

## 주요 기능

- 디스코드의 여러 한글·영문 복사 형식에서 작성자와 시간 제거
- `답장`, `수정`, `전달`, `반응 추가하기` 같은 메뉴 문구 제거
- 일반 이모티콘, 커스텀 이모티콘 코드, 멘션 제거
- 한글·영문 화면 지원
- 사용자가 직접 제거 문구 추가 가능
- 제거된 줄과 보존된 메시지 통계 표시
- 결과 복사 및 TXT 다운로드
- 모바일 화면 지원
- 추적 코드와 외부 스크립트 없음

## 개인정보 보호

TexPick은 입력한 채팅을 서버로 전송하거나 저장하지 않습니다.

- 회원가입 없음
- 분석 도구 없음
- 쿠키 없음
- 데이터베이스 없음
- 채팅 내용 저장 없음

작동 방식은 공개된 소스 코드에서 직접 확인할 수 있습니다.

## 로컬에서 실행하기

`index.html`을 브라우저로 열면 됩니다. 별도의 빌드 과정이 없습니다.

로컬 서버를 사용하려면:

```bash
python -m http.server 8080
```

이후 `http://localhost:8080`에 접속합니다.

## 테스트

```bash
npm test
```

테스트 실행에 별도 패키지 설치가 필요하지 않습니다.
