# TexPick user testing

This checklist helps collect useful feedback without sharing private conversations.

## Before testing

- Use a synthetic conversation or replace every real name and message.
- Never send private chat text in an issue, screenshot, or feedback form.
- Record the Discord client: desktop, web, or mobile.
- Record the interface language.

## Five-minute test

1. Open the [live demo](https://texpick.chattools.workers.dev).
2. Copy a short, anonymized Discord conversation.
3. Paste it into TexPick.
4. Check whether names, times, reactions, and interface labels disappeared.
5. Copy the result.
6. Repeat once in light mode and once in dark mode.
7. If possible, repeat on a phone.

## Feedback template

```text
Device:
Browser:
Discord client and language:

What worked:

What was confusing:

Text that should have been removed:

Real message text removed by mistake:

Would you use this again? Why or why not?
```

## Reporting a parser problem

Open a [bug report](https://github.com/hanlworks/TexPick/issues/new?template=bug_report.yml) with synthetic text that preserves the original line structure.
