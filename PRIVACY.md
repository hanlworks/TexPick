# Privacy

Last updated: June 19, 2026

ChatExtractor is designed to process pasted text entirely on the user's device.

## Data collection

The open-source application does not collect, transmit, or store pasted chat content.

The application contains:

- No analytics SDK
- No advertising SDK
- No account system
- No database connection
- No third-party JavaScript
- No tracking cookies

## Local browser storage

The application stores only interface preferences, such as language and light or dark theme, in the browser's local storage. It does not store pasted chat content or extraction results.

## Hosting

The hosted website provider may process standard network information such as IP addresses and request logs under its own policies. The ChatExtractor application does not add chat text to requests.

## Changes

Any future feature that sends user content over a network must be opt-in, clearly disclosed before transmission, and reviewed as a security-sensitive change.
