# Security Policy

## Supported version

The latest deployed version and the default branch are supported.

## Reporting a vulnerability

Please do not include private chat content, credentials, or personal information in a public issue.

For security-sensitive reports, contact the project maintainer privately through the security contact listed in the GitHub repository profile. Include:

- A clear description
- Reproduction steps using synthetic data
- The expected security impact
- Suggested mitigation, if available

## Data handling

ChatExtractor is designed to process text locally in the browser. A change that introduces network transmission, analytics, remote storage, or third-party scripts must be disclosed prominently and reviewed as a security-sensitive change.
