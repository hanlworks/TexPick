채팅 내용

TexPick은 붙여 넣은 채팅과 추출 결과를 사용자의 브라우저 안에서만 처리합니다. 해당 내용은 서버로 전송하거나 저장하지 않습니다.

수집하지 않는 정보

  • 채팅 내용과 추출 결과
  • 회원 정보
  • 광고 또는 행동 분석 데이터
  • 추적 쿠키

브라우저에 저장되는 설정

언어와 밝은·어두운 테마 같은 화면 설정만 브라우저의 로컬 저장소에 보관합니다. 채팅 내용은 보관하지 않습니다.

호스팅

호스팅 제공자는 자체 정책에 따라 IP 주소와 요청 기록 같은 일반적인 네트워크 정보를 처리할 수 있습니다. TexPick은 채팅 내용을 네트워크 요청에 포함하지 않습니다.

오픈소스

실제 데이터 처리 방식은 공개된 소스 코드에서 직접 확인할 수 있습니다.
